# Rebuttal

査読者へのレスポンス（Rebuttal）用ファイルを管理するディレクトリです。

## ファイル構成例

```
rebuttal/
├── reviewer1.md    # Reviewer 1 へのレスポンス
├── reviewer2.md    # Reviewer 2 へのレスポンス
├── reviewer3.md    # Reviewer 3 へのレスポンス
└── additional_experiments/   # 追加実験の結果
```

## テンプレート

```markdown
# Response to Reviewer X

## Concern 1: [タイトル]

> [査読者のコメント引用]

[レスポンス]

## Concern 2: [タイトル]

> [査読者のコメント引用]

[レスポンス]
```
