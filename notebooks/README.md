# notebooks/

データ探索・可視化・失敗分析のための Jupyter ノートブック。

---

## 使い方のルール

1. **探索専用。本番実験はやらない。** 学習・評価の本番実行は必ず `src/` のコードと
   `scripts/` のスクリプト経由で行う。ノートブックで学習を回しても証拠が残らない。
2. **再現性を要する処理はノートブックに置かない。** 確定したロジックは `src/egosurgery/`
   配下のモジュールへ移し、ノートブックからはそれを import して使う。
3. **出力をコミットする前にセルをクリアする。** `.gitignore` で `.ipynb_checkpoints/` は
   除外済みだが、ノートブック本体の重い出力は手動でクリアしてからコミットする。
4. **番号順に役割が決まっている。** 新規ノートブックは連番を付け、目的を以下の表に追記する。
