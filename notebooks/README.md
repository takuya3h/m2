# Notebooks

## 用途

このディレクトリのノートブックは、以下の目的に使います。

- データの探索・可視化
- 失敗事例の分析
- モデル出力の定性的確認
- アイデアの素早いプロトタイピング

## ルール

- **本番の学習処理はノートブックに置かない**
- ノートブックで整理・確認した処理は `src/` または `scripts/` に移行する
- ファイル名は `00_`, `01_`, `02_` のように順序が分かる形式にする

## ファイル構成

| ファイル | 内容 |
|---|---|
| `00_sanity_check.ipynb` | データ・モデル・パイプラインの動作確認 |
| `01_data_exploration.ipynb` | データセットの統計・可視化 |
| `02_failure_analysis.ipynb` | 失敗事例の分析・可視化 |
